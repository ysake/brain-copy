# AGENTS

## コミュニケーション
- 本プロジェクトのコミュニケーションは日本語で行う

## コミットメッセージ
- 何のための修正かが分かる説明文を書く
- 単なる作業内容だけでなく目的や意図を含める
- 1行で収まらない場合は、1行目に要約、2行目以降に補足を書く

### 例
- Improve Phase 0 tracking to reflect completion status
- Document progress tracking file to clarify migration status
- Fix progress tracking line breaks for readability
